# 💰 Kixikila Digital

## 📌 Descrição
O Kixikila Digital é uma aplicação web desenvolvida para modernizar o sistema tradicional de poupança coletiva, permitindo que grupos organizem contribuições financeiras de forma prática, segura e transparente.

A plataforma facilita o controlo de membros, pagamentos e ciclos de distribuição, adaptando o modelo tradicional ao ambiente digital.
